.. impressjs:: samplepage2
   :data-x: 1200
   :data-y: 0
   :data-rotate-y: 40
   :data-scale: 1

   ソースを reStructuredText_ で記述するだけで、
   ご覧のようなダイナミックなHTMLプレゼンテーションを
   作成することができます。


.. impressjs:: samplepage3
   :data-x: 2400
   :data-y: 0
   :data-rotate-y: 90
   :data-scale: 1

   ソースコードのハイライトも可能です。

   .. code-block:: python

      def hello(name):
          print "Hello, %s" % name

   .. code-block:: bash

      echo $text | tr "[:upper:]" "[:lower:]"
