.. impressjs:: installation
   :data-x: -1000
   :data-y: -800
   :data-rotate-y: 30
   :data-scale: 1

   *Install:*

   .. code-block:: bash

      $ easy_install sphinxjp.themes.impressjs


.. impressjs:: setup
   :data-x: 0
   :data-y: -800
   :data-z: -200
   :data-scale: 1

   *setup your* ``conf.py`` *with:*

   .. code-block:: python

      extensions = ['sphinxjp.themecore']
      html_theme = 'sphinxjp.themes.impressjs'


.. impressjs:: make
   :data-x: 1000
   :data-y: -800
   :data-rotate-y: -30
   :data-scale: 1

   *and run:*

   .. code-block:: bash

      $ make html

   then you'll get this page's style HTML output. Enjoy!
